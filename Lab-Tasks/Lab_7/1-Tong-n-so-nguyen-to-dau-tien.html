Tổng n số nguyên tố đầu tiên Download the code template

Nhập số nguyên n sao cho 0 < n ≤ 50 (nhập đến khi nào thỏa điều kiện thì dừng nhập), nếu n không thỏa điều kiện thì nhập lại. Xuất ra màn hình tổng n số nguyên tố đầu tiên.

Input

Số nguyên n.

Output

  • Xuất thông báo "Gia tri vua nhap la <n>, khong hop le. Vui long nhap lai." nếu n không hợp lệ.
  • Xuất chuỗi "Tong <n> so nguyen to dau tien la: <tong>" nếu n hợp lệ.

Ví dụ

INPUT

OUTPUT

3

Tong 3 so nguyen to dau tien la: 10

0

-6

6

Gia tri vua nhap la 0, khong hop le. Vui long nhap lai.

Gia tri vua nhap la -6, khong hop le. Vui long nhap lai.

Tong 6 so nguyen to dau tien la: 41
